# TCL QD-Mini LED 技术白皮书

## 一、技术概述

TCL QD-Mini LED技术是量子点技术与Mini LED背光的完美结合，代表了当前液晶显示技术的最高水平。该技术将Mini LED背光作为激发源，配合量子点广色域技术，实现了精准的分区控光和广色域显示。

## 二、核心技术原理

### 2.1 QD-Mini LED背光技术

QD-Mini LED将Mini LED背光与量子点技术完美结合，每个LED灯珠尺寸仅为传统LED的1/100。

**技术特点：**
- 灯珠尺寸极小，实现精细分区控光
- 配合量子点广色域技术，实现百万级精细控光
- 157% BT.709广色域覆盖

### 2.2 万象分区控光技术

万象分区控光技术实现了对每个背光分区的独立控制。

**分区数量对比：**
| 系列     | 98英寸 | 85英寸 | 75英寸 | 65英寸 |

| Q10L Pro | 6042   | 5184   | 4032   | 3024   |
| Q10L     | 3840   | 2880   | 2176   | 1680   |

| X11H     | 14112  | 10368  | -      | -      |
| C12H     | 2160   | 2304   | 2160   | -      |

### 2.3 蝶翼华曜屏

采用纳米级仿生科技，在液晶分子上增加聚酰亚胺。

**技术优势：**
- 有效提升对比度
- 去除画面光晕及环境光干扰
- 呈现更加清晰的惊艳画质

### 2.4 量子点技术(QD)

量子点技术通过改变量子点的尺寸来精确控制发光颜色。

**技术特点：**
- 广色域覆盖：157% BT.709
- 色彩还原准确
- 10.7亿级色彩显示

## 附录：技术术语解释

### QD-Mini LED
量子点Mini LED，将量子点技术与Mini LED背光相结合的显示技术。

### Mini LED
迷你发光二极管，灯珠尺寸远小于传统LED。

### 量子点(QD)
一种纳米级半导体材料，通过改变尺寸来控制发光颜色。

### 分区控光
将背光分成多个独立区域，每个区域可独立调节亮度。

### HDR
高动态范围，能够呈现更丰富的明暗细节。

### XDR
极致动态范围，TCL特有的高动态范围技术。

### DCI-P3
数字影院标准色域，电影级广色域标准。

### BT.709
高清电视标准色域，传统电视色域标准。

### BT.2020
超高清电视标准色域，下一代广色域标准。
