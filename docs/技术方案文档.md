# TCLProductAgent - 智能产品咨询助手技术方案

## 一、项目概述

### 1.1 项目名称
**TCLProductAgent** - TCL智能产品咨询助手

### 1.2 项目背景
TCL作为全球领先的消费电子品牌，拥有电视、空调、冰箱、洗衣机等丰富的产品线。消费者在选购产品时面临以下痛点：
- 产品型号众多，参数复杂，难以理解
- 不清楚哪款产品最适合自己需求
- 产品说明书晦涩难懂，查询不便
- 官网客服响应慢，门店咨询成本高

本项目旨在构建一个基于AI Agent的智能产品咨询系统，帮助消费者快速了解TCL产品、获取个性化推荐、解答使用疑问。

### 1.3 核心价值
- **提升用户体验**：24/7即时响应，智能问答
- **降低服务成本**：自动化客服，减少人工成本
- **促进销售转化**：精准产品推荐，缩短决策周期
- **展示AI能力**：体现TCL智能化转型成果

### 1.4 应用场景
1. **官网在线客服**：嵌入TCL官网，替代传统客服
2. **门店智能助手**：线下门店配备智能咨询终端
3. **微信小程序**：移动端产品咨询服务
4. **内部培训工具**：帮助销售人员快速掌握产品知识

## 三、核心功能模块

### 3.1 产品知识问答（核心功能）

**功能描述**：
用户输入产品相关问题，Agent自动检索产品知识库并生成回答。

**技术实现**：
```
用户问题 → 意图识别 → 知识检索(RAG) → 答案生成 → 输出
```

**流程详解**：
1. **问题预处理**：提取关键词，识别产品类别
2. **向量检索**：在ChromaDB中检索相关知识片段
3. **上下文构建**：拼接检索结果 + 用户问题
4. **答案生成**：GLM-4生成自然语言回答
5. **结果优化**：添加产品链接、图片等富文本

**示例对话**：
```
用户：TCL 55Q10G Pro电视的画质怎么样？
Agent：TCL 55Q10G Pro采用QD-Mini LED技术，具有以下画质优势：
      1. 超高对比度：Mini LED背光实现精准控光
      2. 广色域覆盖：量子点技术，色彩还原准确
      3. 4K超清分辨率：细节呈现清晰
      适合观影和游戏，性价比极高！
```

---

### 3.2 智能产品推荐

**功能描述**：
根据用户需求（预算、用途、空间等），推荐最合适的TCL产品。

**技术实现**：
```
用户需求 → 需求解析 → 产品筛选 → 匹配度计算 → 推荐排序 → 输出
```

**推荐算法**：
- **规则匹配**：预算范围 → 价格筛选
- **场景匹配**：用途标签 → 功能筛选
- **相似度计算**：需求描述 vs 产品特点（向量相似度）
- **综合评分**：价格(30%) + 功能(40%) + 口碑(30%)

**示例对话**：
```
用户：我想买一台5000元左右的电视，主要看Netflix和玩游戏
Agent：根据您的需求，为您推荐以下产品：

      🏆 首推：TCL 55Q10G Pro
      - 价格：约5500元
      - 尺寸：55英寸（适合客厅）
      - 优势：Mini LED高对比度，适合观影
      - 游戏性能：120Hz高刷，HDMI 2.1接口

      🥈 备选：TCL 55T7G
      - 价格：约4500元
      - 优势：性价比更高，满足基本需求
```

### 3.3 产品参数对比

**功能描述**：
用户指定多款产品，Agent生成参数对比表格和购买建议。

**技术实现**：
```
产品列表 → 参数提取 → 表格生成 → 差异分析 → 购买建议
```

**对比维度**：
- 基础参数：尺寸、价格、型号
- 技术参数：分辨率、刷新率、核心技术
- 功能特性：智能功能、接口类型
- 适用场景：观影、游戏、办公

**输出格式**：
Markdown表格 + 文字分析 + 推荐结论

---

### 3.4 故障排查指导

**功能描述**：
用户描述产品故障现象，Agent提供排查步骤和解决方案。

**技术实现**：
```
故障描述 → 故障识别 → 方案检索 → 步骤生成 → 输出
```

**安全机制**：
- ⚠️ 不提供专业维修建议（安全风险）
- ✅ 提供自助排查步骤
- ✅ 建议联系官方售后
- ✅ 添加免责声明

## 五、关键技术实现

### 5.1 RAG检索增强生成

**原理**：
结合检索（Retrieval）和生成（Generation），提升回答准确性。

**实现步骤**：
```python
# 1. 文档切片
from langchain.text_splitter import RecursiveCharacterTextSplitter
text_splitter = RecursiveCharacterTextSplitter(
    chunk_size=500,
    chunk_overlap=50
)

# 2. 向量化存储
from langchain_community.embeddings import HuggingFaceEmbeddings
from langchain_community.vectorstores import Chroma
embeddings = HuggingFaceEmbeddings(model_name="text2vec-base-chinese")
vectorstore = Chroma.from_documents(docs, embeddings)

# 3. 检索+生成
from langchain.chains import RetrievalQA
qa_chain = RetrievalQA.from_chain_type(
    llm=llm,
    chain_type="stuff",
    retriever=vectorstore.as_retriever(search_kwargs={"k": 3})
)
```

### 5.4 意图识别与路由

**意图分类**：
```python
def classify_intent(user_input: str) -> str:
    """识别用户意图"""
    intents = {
        "product_query": ["参数", "型号", "怎么样", "好不好"],
        "recommendation": ["推荐", "买什么", "适合", "选择"],
        "comparison": ["对比", "区别", "哪个好", "比较"],
        "troubleshooting": ["故障", "问题", "坏了", "不工作"],
        "tips": ["技巧", "使用", "保养", "如何"]
    }

    # 关键词匹配
    for intent, keywords in intents.items():
        if any(kw in user_input for kw in keywords):
            return intent
    return "general_chat"
```
